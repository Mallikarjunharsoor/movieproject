# Fullstack Movie App (React + Express + MySQL)

## Stack
- Frontend: React + Vite
- Backend: Node.js + Express REST API
- DB: MySQL (Aiven)
- Auth: JWT + bcrypt
- Movies: OMDb API

## Setup
1. Backend:
   - `cd server`
   - `npm install`
   - `npm run dev`
2. Frontend:
   - `cd client`
   - `npm install`
   - `npm run dev`

## URLs
- Frontend: `http://localhost:5173`
- Backend: `http://localhost:5000`
